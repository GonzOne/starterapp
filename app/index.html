Stheory Starter Kit

This starter kit uses Bootstrap 3 for styling, Font Awesome for icons and JQuery for Javascript.

It includes a contact form, basic email template and accompanying sendmail.php script. It utilizes Gulp, a JavaScript task runner file to help ease development.

**NOTE : Default logo size is 50x50 pixels, adjust the CSS based on the size of your logo.
